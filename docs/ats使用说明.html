ats08

ats08 全称08cms automatic tools(08cms 自动化工具) 本来想叫08ats，git仓库不支持数字开头；

安装

  1. 安装git
    • git下载,根据自己的系统自己选择；
    • 安装小乌龟（TortoiseGit）这个可以根据自己的情况，喜欢命令行的可以不用这个，刚开始最好还是用
  2. 安装node
    • node下载

      导出版本库

  3. 小乌龟右键git克隆
  4. url后面填上git@192.168.1.60:ats08.git
  5. 加载putty密钥勾上，后台选择\\ZHONG61\home\目录下自己对应的密钥，例如我的是\\ZHONG61\home\guo.ppk

    安装node插件

  6. 在版本库里右键git bash here打开命令行工具(当然用cmd,powershell也可以)
  7. 执行命令npm start 这个过程要很久，20-30分钟，期间会安装一系列插件
  8. 执行命令npm list,如果没有显示npm ERR表示安装完毕，如果显示再执行cnpm install

    ats结构

  9. .git git的配置文件
  10. gulp gulp的配置文件
    • css gulp时css的配置文件，scss模板，css排序规则
    • tasks 所有的任务
    • base.json gulp的基本配置，包括项目目录，文件作者。。。
    • tasks.json gulp任务的提示信息，用于命令行中的提示，让工具使用更简单
    • utils.js js工具模块，gulp内部使用
  11. node_modules node模板的存放目录，node会自动生成，默认是被git忽略的
  12. src ats的核心部分，css，js，html。。。维护使用主要是这个目录
  13. .gitignore git的忽略文件的配置
  14. gulpfile.js gulp的入口文件，已经将任务分配到tasks文件里了
  15. package.json node模块配置，所有的模块信息及依赖都在这里面

    使用ats之前需要了解的信息

  16. git 会用小乌龟就行
  17. sass 了解下基本方法就行，慢慢深入
  18. seajs 了解下基本方法就行，慢慢深入
  19. bootstrap 主要看下书写风格，ats的代码风格会参照着它的来设计的
  20. markdown 语法简单自己熟悉
